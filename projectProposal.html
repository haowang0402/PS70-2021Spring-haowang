Project Proposal

For my final project, I want to make a smart agricultural system that combines Arduino and statistical methods to optimize productivity. An Arduino with multiple sensors connected to WIFI can keep sending data to my server. By analyzing those data and some research on agricultural academic journals, I can design a system that will have some basic functions of watering and lights based on the results generated from the ML methods. In the long run, I might add soil nutrition analysis to select the type of plants I should plant and even simple harvesting mechanical structures.

  • Step 1: Connect Arduino with sensors and WIFI module. Use data analysis to automate watering optimally. Add mechanical structure for watering.
  • Step 2: Add soil nutrition analysis to select the type of plants I should plant.
  • Step 3: Add mechanical structures for harvesting.
